# IT Help Desk Ticketing System

A responsive web-based ticketing system demonstrating IT help desk WORKFLOWS and ticket management.

## FEATURES

- **Ticket Submission** - Easy form for submitting IT support requests
- **Priority Management** - Set urgency levels (Low, Medium, High)
- **Issue Categorization** - Hardware, Software, Network, Account, Email, Printer, Other
- **Ticket Management** - Mark as resolved, delete, filter by status/priority
- **Real-time Statistics** - Track total, open, and resolved tickets
- **Demo Data** - Automatically loads 8 sample tickets to demonstrate functionality
- **Responsive Design** - Works on desktop, tablet, and mobile

## HOW WE GOT HERE

- HTML5, CSS3, JavaScript (ES6+)
- Object-Oriented Programming
