# IT Help Desk Ticketing System

A simple, responsive web-based ticketing system for IT help desk operations. Built to demonstrate understanding of help desk workflows, ticket management, and front-end development.

## ğŸš€ Getting Started

### Installation

1. Clone this repository:
```bash
git clone https://github.com/alltechnofoul/helpdesk-ticketing-system.git
cd helpdesk-ticketing-system
```

2. Open `index.html` in your web browser

That's it! No dependencies, no build process required.

### Usage

**To Submit a Ticket:**
1. Fill out the form on the left side
2. Provide name, email, category, priority, and description
3. Click "Submit Ticket"

**To Manage Tickets:**
1. View all tickets on the right side
2. Use filters to sort by status or priority
3. Click "Mark as Resolved" to close a ticket
4. Click "Delete" to remove a ticket
